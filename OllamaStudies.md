For MacOS and Linux Distros*
``` Mac & Linux
curl -fsSL https://ollama.com/install.sh | sh
```
Windows CMD or PowerShell
``` Windows
irm https://ollama.com/install.ps1 | iex
```

or Download link with https://ollama.com/download/mac if you use MacOS, https://ollama.com/download/windows if you use Windows, but for Linux users, you must use the terminal, it the only way.
